Herramienta de Transliteración Árabe

Convierte texto árabe a script latino y escucha la pronunciación.

⚠️ Importante: Para escuchar el árabe con buena calidad, necesitas instalar voces árabes en tu sistema. Ver cómo instalar voces
Calidad de audio: Esta herramienta usa las voces de tu sistema operativo. Para mejor calidad:
  • Windows: Configuración → Hora e idioma → Idioma → Árabe → Opciones → Descargar voz
  • Mac: Preferencias → Accesibilidad → Voz → Descargar voz árabe
  • Móvil: Las voces suelen estar preinstaladas
